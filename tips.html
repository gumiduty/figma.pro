Лайфаки для Профі

Інструменти, які економлять 50% робочого часу.

1. Auto Layout (Shift + A)

Це функція, яка відрізняє новачків від сеньйорів. Auto Layout дозволяє створювати динамічні кнопки, списки та таблиці. Якщо ви зміните текст у кнопці, вона автоматично розшириться. Якщо поміняєте місцями пункти меню стрілочками клавіатури, відступи залишаться ідеальними.

Pro Tip: Використовуйте "Absolute Position" всередині автолейауту (іконка прицілу), щоб розмістити елемент (наприклад, бейдж повідомлення) поверх структури.

2. Компоненти та Варіанти

Ніколи не копіюйте кнопки вручну. Створіть Компонент (Ctrl+Alt+K). Тепер, коли ви зміните колір головної кнопки, він зміниться на всіх 50 сторінках вашого проекту. Використовуйте Варіанти (Variants), щоб об'єднати різні стани кнопки (Default, Hover, Disabled) в один логічний елемент.

3. Гарячі клавіші (Must Know)

  • Ctrl + R — Масове перейменування.
  • Shift + 2 — Зум на виділений об'єкт.
  • Ctrl + Alt + C — Копіювати стиль об'єкта.
  • Ctrl + Alt + V — Вставити стиль.

Використання гарячих клавіш прискорює роботу в 3 рази. Спробуйте натиснути Ctrl + /, щоб відкрити швидкий пошук по меню.
